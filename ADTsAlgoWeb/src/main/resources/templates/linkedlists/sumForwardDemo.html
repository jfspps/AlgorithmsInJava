Building a sum expression from nodes in a list

Operands with equal number of digits

Change the positive integer values and click Process. Note that { 0, 1, 2, 3, 4, 5 } is equivalent to 12 + 345, that is, both operands are always three digits long. Similarly, { 0, 1, 2, 0, 0, 5 } is equivalent to 12 + 5.

Operands with unqeual number of digits

Change the positive integer values and click Process. Note that { 1, 1, 2, 3, 4 } is equivalent to 112 + 34, that is, the first operand is longer than the second. Regarding zeros, { 0, 1, 2, 1, 5 } is equivalent to 12 + 15 and { 0, 1, 2, 0, 5 } is equivalent to 12 + 5.
