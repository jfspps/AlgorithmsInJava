Finding the kth-to-last node of a singly linked list

This algorithm instructs a pointer to traverse a singly linked list until it finds the Kth to last node. 

        
        private LinkedList<T> list;

        public FindKthToLast(LinkedList<T> list) {
            this.list = list;
        }

Indices are zero-based. The last node is represented by the pointer index, where index = length - 1 and length = length of the list. The second to last element would be the length - 2 index. The kth-to-last element is thus represented by index = length - k. The pointer is incremented while (index < length - k). 

        
        // time complexity is O(n)
        public LinkedList<T> findKthToLast(int k){
            if (list == null){
                System.out.println("List is empty");
                return null;
            }

            // length is decremented later
            int length = list.length();

            if (k > length){
                System.out.println("Kth parameter is larger than the list length");
                return null;
            }

            int nodeIndex = 0;
            LinkedList<T> currentNode = list;
            // k decreases from length to 1 as list traversed from head node
            // start from the beginning, increment nodeIndex = 0 until (nodeIndex = length - k)
            while (nodeIndex < length - k && currentNode.next != null){
                currentNode = currentNode.next;
                nodeIndex++;
            }

            return currentNode;
        }
