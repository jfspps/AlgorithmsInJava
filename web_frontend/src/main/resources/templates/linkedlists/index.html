Linked lists

The sub-topics covered are:

  • Custom implementation of a Linked list
  • List to "set": removing duplicates from a linked list
  • Finding the kth-to-last element of a linked list
  • Partitioning a linked list at linkedList x, such that all nodes that are less than or equal to x are located to the left
  • Building a sum expression from the digits stored in a linked list and then performing summation. The digits are read from the linked list left-to-right.
  • Check if a linked list spells out a palindrome
  • Return a reference (not value) to the intersection point of two singly linked lists, if any
  • Return a reference to the linkedList where a loop begins in one singly linked list, if any
