Algorithms in Java

This project presents the code with runnable examples of commonly applied ADTs and algorithms. Most of the examples are my solutions to the problems from "Cracking the Code Interview". The topics covered are:

  • Arrays, StringBuilder and Hashtables
  • Linked lists
  • Stacks and queues
  • Trees and graphs
  • Bits and bytes
  • OOP and design patterns
  • Recursion and Memoization
  • Searching and sorting methods
  • Concurrent programming
